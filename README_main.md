## Fraud Detection

**[→ View Project](./Fraud-detection/)**

An AI-powered fraud detection agent that reasons about social engineering attacks using behavioral science frameworks — not just keyword matching.

**What it does:**
- Analyzes conversation transcripts and case descriptions
- Identifies psychological manipulation tactics (Cialdini, BITE model)
- Maps the case to a scam lifecycle phase
- Outputs risk score, red flags, and victim-appropriate intervention language

**Why it matters:** Modern scams exploit human psychology, not technical vulnerabilities. Detection systems need to reason about *tactics*, not just *patterns* — this agent demonstrates that approach.

## Fintech Risk Analytics

**[→ View Project](./Fintech-risk/)**

A two-part analytics toolkit covering four risk domains common to fintech and banking fraud operations.

**Interactive Dashboard** — Upload any transaction CSV, get instant analysis:
- Auto-detects column schema
- Risk scores every transaction using a rule + ML ensemble
- Visualizes amount distribution, hourly patterns, risk tiers
- AI generates a structured investigation report

**Jupyter Notebook** — Full analytical pipeline across four domains:
- Transaction fraud detection (credit card / payment)
- Merchant risk scoring and portfolio analysis
- AML network analysis and SAR trigger logic
- Credit application fraud scorecard
